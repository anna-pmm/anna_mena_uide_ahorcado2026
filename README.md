## Problemática
Se requiere desarrollar un sistema interactivo que permita simular el juego del Ahorcado mediante programación estructurada. El programa debe seleccionar una palabra de manera aleatoria, permitir al usuario ingresar letras para intentar adivinarla y controlar el número de intentos disponibles. Además, el sistema debe actualizar el progreso del jugador, validar las entradas y determinar correctamente las condiciones de victoria o derrota.
El reto principal consiste en coordinar correctamente las estructuras condicionales, repetitivas y el uso de listas para asegurar que el juego funcione de manera coherente hasta su finalización.


## Objetivo
Desarrollar un programa en Python que permita aplicar variables, condicionales, bucles, listas y funciones para resolver un problema interactivo.


## Descripción del Software
- El sistema selecciona una palabra aleatoria desde una lista y entrega como pista la primera y última letra de la palabra.
- El usuario debe ingresar letras para intentar adivinarla.
- El juego controla el número de intentos. 
- Finaliza cuando se completa la palabra o se agotan los intentos.


## Funcionalidades
- Selección aleatoria de palabra.
- Visualización del progreso.
- Control de intentos.
- Condición de victoria y derrota.
- Uso de funciones para modularizar el programa.


## Resultados
El programa permite simular el juego completo del ahorcado utilizando estructuras vistas en las cuatro unidades de la asignatura.


## Limitaciones
- Funciona únicamente en consola.
- No incluye interfaz gráfica.
- No almacena estadísticas de partidas.


## Conclusión
El desarrollo del juego del Ahorcado permitió implementar de manera integrada estructuras condicionales, repetitivas, listas y funciones, garantizando el control del flujo del programa hasta su finalización. 

El proyecto demuestra el uso práctico de la lógica de programación para resolver un problema interactivo en un entorno de consola. Por lo tanto, este proyecto integra los contenidos desarrollados en las cuatro unidades de la asignatura.
